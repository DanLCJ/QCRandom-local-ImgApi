调用地址:http://xxx.com/wallpaper.php
调用参数:return = http / https / url / json / img / xml / base64Code / base64Img
json：返回标准json数据(图片地址)
url：直接返回图片链接
img：直接显示图片不返回图片链接
http: 返回HTTP图片链接
https: 返回HTTPS图片链接
xml: 返回标准XML数据(图片地址)
base64Code：返回标准Base64图码数据
base64Img：返回标准Base64图片
